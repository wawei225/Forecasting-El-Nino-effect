Comments

  • ACF and PACF suggests an AR(2) model for the following reason
    • ACF appears to be decreasing quickly which suggest a possibility for an AR process
    • PACF has zero partial autocorrelation coefficients after lag 3 (PACF will usually have non-zero partial autocorrelation coefficients for lags up to the order of the model)
  • We reject the null hypothesis of ADF test and conclude that the series is stationary
  • We fail to reject the null hypothesis of KPSS test and conclude that the series is stationary.

Comments

The null hypothesis of Shapiro-wilk test is: H0: the studentized residuals are normally distributed. The null hypothesis of Run test is: H0: the studentized residuals are independent, i.e., the series is random. The null hypothesis of Box-Ljung test is: H0: the residuals are not correlated.

  • Based on above testing and plots, we conclude the student residuals for AR(2) and ARIMA(2,0,3) are normally distributed and independent and correlated.
    • We should keep in mind that the correlation can potentially lead an issue while we analyzing the model, even the normality and independent assumptions are satisfied.

# comparing forecast model and selection

#  

# set up -----------------------------------------------------------------------

m <- 2              # two different ARIMA models
k <- 36             # suppose we want to make 24 forecast comparisons.  ## need to change this depends on what you want 
n.obs <- length(el.nino.ts)
n.end <- n.obs-k    # Only use n.end observations for the first forecast, which starts at may 2016.
el.nino.ts.trun <- window(el.nino.ts,start=c(2016,5))       # the actual series from 2016 to 2019
pred <- matrix(rep(0,k * m),k,m)        # create a empty matrix that will be filled by forecast values

# generate two sequences of forecasts from two models
# step 1: use a subsample to build the model, and then do 1-period ahead forecast,
# step 2: increase the length of the subsample by 1 period and repeat step 1
for(i in 1:k){
  x <- el.nino.ts[1:n.end+i-1]
  model1.tmp <- Arima(x,order=c(2,0,0), method="ML")        # Candidate Model 1
  pred[i,1] <- forecast(model1.tmp,h=1)$mean[1]         # h=1 means 1-period ahead forecast; ## mean is the point forecast
  model2.tmp <- Arima(x,order=c(2,0,3), method="ML")        # Candidate Model 2
  pred[i,2] <- forecast(model2.tmp,h=1)$mean[1]         # h=1 means 1-period ahead forecast
}
pred.ts <- ts(pred, start=c(2016,5), frequency=12)

# dm test ----------------------------------------------------------------------

# Run the modified DM test on these 1-period ahead forecasts from two models
e1 <- el.nino.ts[(n.end+1):n.obs]-pred[,1]
e2 <- el.nino.ts[(n.end+1):n.obs]-pred[,2]
dm.test(e1,e2,h=1,alternative="less",power=2)       # model 1 is better
## 
##  Diebold-Mariano Test
##

## 0.5856

  • Based on the plots and Diebold-Mariano test, both models have similar forecasting power. Therefore, we choose the one with less order which is ARIMA(2,0,0)
